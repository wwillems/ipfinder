The freegeoip.net offers a public RESTful Web Service for searching geolocation of IP addresses and host names.

This system has an internal database with geolocation information, which is queried via the web service.

There's no magic or tricky calculations, it's a database. Thus, although it's very accurate, don't expect it to be perfect.
